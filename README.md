
# My Diary App

My Diary is a personal diary for every day. It is 
a diary with mood tracker to explore your emotions and add photos, videos and music files to entries.


## Tech Stack

- MVVM

- Kotlin Flow

- Room

- Kotlin Extensions

- Hilt

- Content Provider


# Featues
#### 1.  Search
#### 2.  Add Entry
#### 3.  Update Entry
#### 4.  Delete Entry
#### 5.  Add Mood
#### 6.  Add Media
#### 7.  Search
